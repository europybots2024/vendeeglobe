# vendeeglobe

Around the world sailing race

## TL;DR

Get started with:

```
conda create -n <NAME> -c conda-forge python=3.10
conda activate <NAME>
git clone https://github.com/europybots2024/vendeeglobe.git
cd vendeeglobe/
python -m pip install -e .
cd run/
python test.py
```
